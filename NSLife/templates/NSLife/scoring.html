IPPT Score Calculator

Points are calculated based on how well you do on the 3 IPPT stations:

  • Push Up Station
  • Sit Up Station
  • 2.4km Run Station

For Push Up and Sit Up Station, you should count the max reps done in one minute.

For 2.4km Run Station, you should take the time taken for you to complete a 2.4km run.

Points are calibrated to your age group at each IPPT station, with the age groups being calibrated within 3-year bands.
